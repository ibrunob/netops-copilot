# Architecture decision records

Record decisions that affect the production architecture here once they are made.
Use a short immutable file name such as `0001-postgres-rls.md`, state the context,
decision, consequences, and supersession status, and do not rewrite accepted
decisions without a new superseding ADR.

The target architecture is documented in [`../ARCHITECTURE.md`](../ARCHITECTURE.md).
No ADR has been accepted in this directory yet.
